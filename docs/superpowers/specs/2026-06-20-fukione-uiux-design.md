## 1. Design principles (global)

These apply to every screen below.

1. **A lead door is always within thumb's reach.** On mobile, a sticky bottom action bar keeps
   a primary CTA + Zalo visible at all times, regardless of scroll position.
2. **One primary action per screen.** Each screen has exactly one primary CTA so the visitor is
   never forced to choose between competing calls to action. Secondary actions are visually
   subordinate.
3. **Always offer "skip the form, talk now."** Next to every form CTA there is a bypass —
   "hoặc 💬 tư vấn ngay / ☎ gọi" — that goes straight to Zalo/hotline with no data entry
   (use cases UC-10, UC-16). Some buyers want a human immediately; never block them behind a form.
4. **Show trust early.** Real Hanoi showroom (NAP + map), "tư vấn miễn phí", and real project
   photos appear high on the page. Social proof (completed projects) is the strongest closing
   tool in this industry.
5. **Every number carries a disclaimer.** Any price or m² estimate is always shown with
   "*giá cuối cần khảo sát*" — transparent, and it opens the door for sales to consult/upsell.
6. **Read → lead.** Every content page (blog, project, collection) embeds at least one CTA that
   funnels the reader into lead capture.

## 2. Global UX framework

Repeated "chrome" present on every page.

### Mobile (default)

```
┌─────────────────────────┐
│ ☰   FUKIONE        ☎ 💬 │ ← sticky top bar: menu | logo | quick call + Zalo
├─────────────────────────┤
│                         │
│      PAGE CONTENT        │
│      (vertical scroll)   │
│                         │
├─────────────────────────┤
│ [ 🧮 Tính chi phí ]  💬 │ ← sticky bottom ACTION BAR (always visible)
└─────────────────────────┘
   primary button changes by page context:
     - Home:           [Tính chi phí]
     - Product page:   [Nhận báo giá]
     - Project page:   [Đặt lịch khảo sát]
```

| Component | Decision | Rationale |
|---|---|---|
| **Sticky top bar** | Logo + hamburger + 2 quick icons (☎ call, 💬 Zalo) | Call/Zalo are how Vietnamese buyers close — always one tap away |
| **Sticky bottom action bar** (mobile) | One primary button (context-dependent) + Zalo | Thumb zone; the lead door never leaves the screen |
| **Drawer menu** | Sản phẩm · Bộ sưu tập · Dự án · Blog · Giới thiệu · Liên hệ | Matches the sitemap; minimal |
| **Footer** | NAP (Name–Address–Phone) + business hours + map link + Zalo | Required for Local SEO + trust signal |
| **Desktop (>768px)** | Full horizontal nav; **no** bottom bar; primary CTA in header + inline | Bottom bar not needed on wide viewports |

## 4. User flows (lead capture)

The heart of the product: **4 entry doors → converge on one lead mechanism → one thank-you.**

```
ENTRY (Google / Zalo / Maps)
   │        │           │            │
   ▼        ▼           ▼            ▼
[Home]   [Catalog]  [Project/Blog] [Zalo] ──► sales chat (leaves site)
   │        │           │
   │        ▼           │
   │   [Product detail] │
   │        │           │
   └────────┼───────────┘
            ▼
   ┌─────────────────────────────────┐
   │   4 LEAD DOORS (CTA)             │
   │   🧮 Tính chi phí (calculator)   │
   │   📝 Nhận báo giá (quote)         │
   │   📅 Đặt lịch khảo sát (survey)   │
   │   💬 Zalo (direct consult)        │
   └────────────────┬────────────────┘
                    ▼
        [Short form: Tên* + SĐT*]   ← auto-attaches product/m²/estimate if from calculator
                    │ submit
                    ▼
        ① Save Lead to DB (status=new)   ← always runs first; lead is never lost (NFR-REL-01)
                    │
          ┌─────────┴──────────┐
          ▼ (background)       ▼
   ② Email sales         [Thank-you page /cam-on]
   (on error: retry,      "Sales will call in ~15'"
    lead stays safe)      + Zalo + hotline; GA4 conversion fires
```

### Two golden paths (basis for future E2E tests)

```
PATH A — Calculator-led (primary):
[Home] →tap "Tính chi phí"→ [Calculator]
  enter 25m² + select product + ☑ install → see ESTIMATE ~14M
  →tap "Nhận báo giá chính xác"→ [Form] (product/m²/price pre-filled)
  → submit → [/cam-on]

PATH B — Catalog-led:
[Catalog] →filter(color/thickness/waterproof/price/room)→ [Product detail]
  → two branches: "Tính chi phí"(→Path A, product pre-selected) | "Nhận báo giá"(→Form)
  → [/cam-on]
```

### Bypass rule (always available)
Every conversion point exposes "hoặc 💬 tư vấn ngay / ☎ gọi" that jumps straight to
Zalo/hotline without any form — for buyers who want a human immediately.

## 6. Content/trust templates (light)

Shared rule for this group: **read → lead** — every page embeds at least one CTA
("Tính chi phí" or "Đặt khảo sát"), mid-content and at the end.

### T1 — List template (`/blog`, `/du-an`)
```
│ Title + (blog: tag chips)
│ 2-col card grid → [Tải thêm]
│ CTA strip mid/bottom
│ Footer NAP
```

### T2 — Article template (`/blog/[slug]`)
```
│ Cover image · H1 · date
│ Rich-text body
│ ┌─ inline CTA "Tính chi phí ngay" ─┐  ← mid-article
│ Related posts → · end CTA strip
   SEO: Article + BreadcrumbList + FAQPage schema
```

### T3 — Project detail (`/du-an/[slug]`) — strongest closing tool
```
│ [Gallery of real install photos] swipe
│ "Thi công sàn gỗ — [district]" · area, duration
│ SP ĐÃ DÙNG [product cards]→  ← links to product detail
│ [ Tôi cũng muốn tư vấn ]      ← CTA
```

### T4 — Collection (`/bo-suu-tap/[slug]`)
```
│ [concept image "Phòng khách"] · short concept copy
│ SP TRONG BỘ [product grid]  ← pre-filtered by concept

```

### T5 — About (`/ve-chung-toi`)
```
│ Brand hero · story / commitment
│ Numbers: X years · Y projects  ← trust
│ Showroom HN [photo + map]
│ 4-step process
│ [ Đặt lịch tham quan ]  ← CTA
```

### T6 — Contact (`/lien-he`)
```
│ ☎ Hotline  💬 Zalo  ← one tap
│ 📍 Showroom address (NAP) · 🕐 hours
│ [ GOOGLE MAP embed ]  ← Local SEO
│ Short contact form
   SEO: LocalBusiness schema
```

### T7 — 404 / 500 (friendly)
```
│ 🪵 "Trang không tồn tại"
│ [ Về trang chủ ] [ Xem sản phẩm ]
│ hoặc 💬 Zalo tư vấn
```

### Cross-cutting content rules
1. **Read → lead:** mid + end CTAs on every content page.
2. **Cross-linking:** Project ↔ products used ↔ Collection → keeps the visitor on-site, good for SEO.
3. **Per-page Schema.org:** Article (blog), LocalBusiness (contact), Product (product detail), Breadcrumb/FAQ.
4. **Consistent NAP** in the footer on every page.
